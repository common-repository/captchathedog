== Description ==

Photo Captcha with user defined images to reduce the spambot prize/payload to one website at a time. Creating a new captcha puzzle one new image at a time. Building a new spambot for each website with unique user defined images does not make financial sense for the spammer. 

A captcha with "Which one of these things is not like the other?" puzzle.



Version history:
- 1 may 2009: created
- 5 may 2009: public released
- 7 may 2009: admin does not need to use captcha, catches php.ini file error





== Installation ==

   1. Upload `captchathedog` to the `/wp-content/plugins/captchathedog/` directory
or 1. Activate the plugin through the 'Plugins' menu in WordPress


Some servers need the following modification to their pho.ini:
For example /home/[userID]/etc/php.ini

Add these lines:
allow_url_include = On
allow_url_fopen = On

I hope this helps..
